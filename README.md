Graphite is an advanced educational tool that transforms 2D engineering drawings and sketches into interactive 3D models. Powered by **Google Gemini**, it helps students and engineers visualize complex component assemblies, understand construction steps, and generate technical documentation.

---

## üöÄ Key Features

### üì∏ **Smart Scan & Generation**
- **Camera Capture**: Integrated camera support with a custom viewfinder for capturing physical drawings.
- **AI Validation**: Automatically verifies if an image is a valid engineering drawing before processing.
- **3D Reconstruction**: Uses Gemini 1.5 Flash and Gemini 3 Preview to interpret orthogonal/isometric views and generate Constructive Solid Geometry (CSG) models.

### üèóÔ∏è **Interactive Studio**
- **Dual View Modes**: Switch seamlessly between **Assembled** and **Exploded (Split)** views to understand part relationships.
- **Wireframe & Dimensions**: Toggle technical overlays for precise inspection.
- **Unit Support**: Check dimensions in Millimeters (mm) or Centimeters (cm).

### ü§ñ **AI Engineering Tutor**
- **Context-Aware Chat**: Ask questions about the specific model being viewed. The AI understands the part list, dimensions, and geometry.
- **Markdown Support**: Chat responses are formatted with clear bullet points, bold key terms, and structured paragraphs.
- **Audio Construction Guide**: Listen to step-by-step assembly instructions generated by natural-sounding AI (Gemini 2.0 Flash Audio).

### üìÑ **Professional Exports**
- **Smart PDF Reports**: Export "Assembled", "Split", or "Combined" technical reports.
- **Auto-Framing**: Intelligent camera logic ensures the model is perfectly scaled and centered in the PDF, regardless of your screen zoom.
- **Detailed Instructions**: Automatically appends a parts list and step-by-step construction guide to the export.

## ‚ö° Getting Started

### Prerequisites
- Node.js (v18 or higher)
- A Google Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/graphite.git
   cd graphite
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env.local` file in the root directory and add your API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   # Or if using the server-side proxy pattern
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open your browser to `http://localhost:5173` (or the IP shown in terminal for mobile testing).

## üì± Mobile Usage
To use the Camera feature on mobile:
1. Ensure your dev machine and mobile device are on the same Wi-Fi.
2. The project is configured to allow HTTPS for camera permissions.
3. Access via `https://<YOUR_LOCAL_IP>:5173`.
4. Accept the self-signed certificate warning if running locally.
